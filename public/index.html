How It Works

  1. Create a channel point redemption on your Twitch channel named "Song Request" (or whatever you've configured in your .env file)
  2. Make sure to enable the "Require viewer to enter text" option for the redemption
  3. Viewers can redeem channel points and enter a song name, Spotify URL, or Spotify URI in the message
  4. The bot will automatically add the requested song to your Spotify queue

Setup Instructions

  1. Create a Twitch Developer Application at https://dev.twitch.tv/console/apps
  2. Create a Spotify Developer Application at https://developer.spotify.com/dashboard
  3. Add the redirect URI: http://127.0.0.1:8888/callback to your Spotify app
  4. Copy your Client IDs and Client Secrets to the .env file
  5. Make sure Spotify is running on your device
